WINAPI: is a calling convention involving how machine code is generated to place function call args on stack. Most winAPI functions are called by it.11

winMain: The name of the entry function.11

LPSTR/PSTR: Pointer to character string.11

hInstance: The parameter is used to save the instance handle.11

hPrevInstance: Used by  app to check if instances of self were running.11

MessageBox: Displays a message box with not much versatility. The first argument to it is normally a window handle. The second argument is the text string that appears in the body of the message box, and the third argument is the text string that appears in the caption bar of the message box. The fourth argument to MessageBox can be a combination of constants beginning with the prefix MB_ that are defined in WINUSER.H .12
